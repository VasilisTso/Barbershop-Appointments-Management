# Barbershop-Appointments-Management
This is a project for the university class Special topics in Software Technology. Its a full stack web app about managing appointments at a barbershop made using React, Node.js, SQLite. Since its a CRUD business app
, it has role-based features. In more detail, the user can see the services, book appointments and see only his appointments. The admin of the app can create, edit and delete services, and see all appointments made by all the users
